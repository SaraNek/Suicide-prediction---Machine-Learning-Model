## 📂 Dataset

* The dataset was collected through a small-scale survey (~100+ participants)
* Questions were designed to capture behavioral responses and attitudes using humor-based prompts
* The survey included indirect indicators rather than explicit clinical questions

⚠️ **Limitations:**

* Small dataset size
* Informal and non-clinical data collection method
* Results should not be generalized to real-world populations

---

## ⚙️ Approach

1. Data Collection through survey responses
2. Data Cleaning and Encoding
3. Exploratory Data Analysis
4. Training a Random Forest model
5. Evaluating patterns and feature importance

---

## 📈 Results & Insights

* The model was able to identify patterns within the dataset
* Certain behavioral responses showed higher influence on predictions

⚠️ These results are exploratory and not suitable for real-world decision-making
